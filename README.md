# El archivo del rey pálido
Un proyecto de Ingeniería de Datos para el registro, monitoreo y análisis de Hollownest en la ausencia del Rey Pálido.

## Introducción
El Rey Pálido nos ha pedido que en su ausencia actualicemos los archivos de Hollownest para digitalizarlos y para llevar un registro de las vasijas que ha creado para combatir la infección.
Nuestro objetivo principal es digitalizar y centralizar los archivos históricos, así como implementar un sistema de monitoreo en tiempo real para:
- Registrar el Estatus de las Vasijas: Documentar la creación y el estado actual de las Vasijas destinadas a contener la Infección (jugadores).
- Preservar la Historia: Crear un repositorio consultable del vasto conocimiento de Hallownest.
Este proyecto actúa como el Núcleo de Datos para el futuro del reino, transformando pergaminos antiguos y observaciones dispersas en una fuente de verdad estructurada y analítica.
